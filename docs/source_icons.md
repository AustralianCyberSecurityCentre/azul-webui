Azul has a set of predefined icons available for use for annotating sources with.

Each source has one icon, but icons can be shared between

Available icons:

- gears
- ambulance
- book
- bug
- search
- trash
- trash-alt
- cloud-download
- cloud-download-alt
- binoculars
- heartbeat
- shield
- shield-alt
- screwdriver-wrench
- bell
- bomb

These all are Font Awesome icons, but future icons may come from alternative
sources.
